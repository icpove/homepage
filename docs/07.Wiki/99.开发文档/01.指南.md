## 基础

参与本站的开发需掌握下面这些知识：

- 熟练掌握 markdown 、yaml 语法
- 会使用终端（命令行），会使用 git
- 会阅读文档，搜索文档，正确使用 Baidu、Bing、Google 等搜索引擎
- 会 VuePress 的基本使用和默认主题的基本配置
- 会使用 GitHub 团队协作开发项目

掌握以上基础知识，你就可以愉快的进行 Wiki 和 blog 的撰写了，另外掌握 VUE 你就可以进行功能性开发了。
